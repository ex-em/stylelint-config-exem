# stylelint-config-exem
- stylelint-config-exem은 exem css 컨벤션 lint 룰 셋입니다.

## Install
```bash
npm install stylelint stylelint-config-exem --save-dev
```

## Usage
`stylelint.config.js` 파일을 루트 디렉토리에 생성해주세요
```javascript
module.exports = {
  extends: 'stylelint-config-exem',
  rules: {
    // 프로젝트 적용할 룰
  },
};
```
